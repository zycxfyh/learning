# 🎯 任务书：第一步 - 打造你的 Tensor 容器

## 目标
你要在 `step1_tensor_lab.c` 中完成 Tensor 的定义与内存管理。这是所有深度学习框架（如 PyTorch/TensorFlow）最底层的逻辑。

## 任务清单
1. **定义结构体**：让 `struct Tensor` 能够记住自己的形状（几行几列）并持有一块内存地址。
2. **内存申请 (malloc)**：
   - 第一次申请：为 `struct Tensor` 申请空间。
   - 第二次申请：为 `rows * cols` 个 `float` 数字申请空间。
3. **内存释放 (free)**：
   - 顺序很重要！必须先把“里面的数据”丢掉，再把“外面的壳子”丢掉。否则你会找不到数据在哪，造成内存泄漏。

## 关键知识
- `malloc(size)`: 向系统要钱（内存）。
- `sizeof(type)`: 算出某种类型占多少字节。
- `->`: 这是你上一课学过的，通过指针访问结构体成员。
