## 1. 冒泡排序 (Bubble Sort) - **必考算法**
逻辑：两两比较，大的往后走。
```c
void bubble_sort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j+1]) {
                int temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }
}
```

## 2. 字符串常见操作
### A. 统计字母/数字个数
```c
int count = 0;
while (str[i] != '\0') {
    if (str[i] >= 'a' && str[i] <= 'z') count++;
    i++;
}
```
### B. 字符串逆序 (经典)
```c
for (int i = 0, j = strlen(s)-1; i < j; i++, j--) {
    char t = s[i]; s[i] = s[j]; s[j] = t;
}
```

## 3. 结构体与排序 (综合大题)
题目通常是：定义学生结构体，输入N个学生数据，按成绩排序。
- **要点**: 交换时，要交换**整个结构体变量**，而不仅仅是分数。

## 4. 指针与数组 (填空/程序浏览)
- `*p++` : 取值，然后指针后移。
- `*(p+i)` 等价于 `arr[i]`。

## 🚀 你的第一个“正式”项目：`my_wc`
在考试之余，你可以尝试编写这个简化版的 `wc` (Word Count)。它只需要用到文件读取、循环和判断。

**功能**: 统计文件的行数、单词数和字符数。
**挑战点**: 怎么判断一个“单词”开始了？（通常是遇到空格或换行）。
