## 📊 题目元信息
- **难度等级**: ⭐⭐⭐⭐⭐⭐ (5.5-5.8/10)
- **年份**: 2024 ICPC World Finals

- **核心技术**: 最优化 + 贪心算法

---

## 🎯 问题描述

**故事背景**:
广告牌公平分配问题。有n个赞助商和一块长度为L的广告牌，每个赞助商对广告牌的不同部分有不同的估值函数（连续分段线性函数）。

**核心任务**:
将广告牌分成n个连续的段落，每个赞助商分配一段，使得每个赞助商获得的估值至少为其总估值的1/n。

### 📥 输入规格
```
第一行: n L
接下来 n 行: 每行描述赞助商的估值函数
```

### 📤 输出规格
- 分配方案或 `IMPOSSIBLE`

---

## 🔍 难点剖析

### 🚨 三大挑战
1. **连续函数积分**: 分段线性函数的精确积分计算
2. **公平性保证**: 多目标优化下的公平分配
3. **贪心策略设计**: 最优分割点的选择算法

---

## 🧠 解题策略

### 🎯 总体思路
**贪心算法 + 函数积分 + 二分搜索**

### 📋 详细步骤

#### 阶段1: 函数预处理
- 解析分段线性函数
- 计算积分值

#### 阶段2: 贪心分配
- 选择最需要的位置
- 更新剩余空间

#### 阶段3: 公平性验证
- 检查分配结果
- 调整分割点
